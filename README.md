

<h1 align="center">DietApp</h1>

## Description

### This android application calculates the body mass index of the user and recommends a diet for 30 days. Recipes and youtube video of recommended dishes are also shown.




## Built With

- Java
- Firebase
- Picasso
- YoutubePlayer Api
